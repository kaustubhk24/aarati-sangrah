# श्री नवनाग स्तोत्र 

अनन्तं वासुकिं शेषं पद्मनाभं च कम्बलं

शन्खपालं ध्रूतराष्ट्रं च तक्षकं कालियं तथा



एतानि नव नामानि नागानाम च महात्मनं

सायमकाले पठेन्नीत्यं प्रातक्काले विशेषतः

तस्य विषभयं नास्ति सर्वत्र विजयी भवेत
